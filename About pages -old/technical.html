Technical Information

This site is generated with Versioning Machine (with significant new development by James Cummings and Theodore Koterwas at InfoDev, a division of IT Services at the University of Oxford The text-transcriptions are encoded in XML according to the TEI guidelines (P5) transformed to html with XSLT. The XML file is available at the project github. Image files are displayed using OpenLayers
